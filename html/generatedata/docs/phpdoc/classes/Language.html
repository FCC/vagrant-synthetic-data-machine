    This class contains all functionality relating to the current selected language.

    For code relating to all available translations, see the Translation class.

    "fr" in "fr.php") as the first param. But to keep things as simple (and still as loosely coupled) as possible, there's an optional second param which will override the value passed in param #1. This overrides the value first based on whatever's in sessions and secondly, whatever is in the query param. The query param always takes precedence.
